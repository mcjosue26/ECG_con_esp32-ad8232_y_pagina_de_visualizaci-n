ECG Hardware





Hardware del proyecto


En esta sección se describe el hardware y componentes que se utilizaron en el proyecto para la adquisicion de las señales del electrocardiograma.



Ad8232 Ecg Módulo Monitor De Pulso Cardíaco




El corazón de este módulo es un circuito integrado AD8232 que integra en un solo circuito el front-end analógico y permite medir la actividad eléctrica del corazón. Está diseñado para extraer, amplificar y filtrar pequeñas señales biopotenciales en presencia de condiciones ruidosas, así como aquellas creadas por el movimiento o la colocación de electrodos.



Microcontrolador ESP32




El módulo ESP32 es una solución de Wi-Fi/Bluetooth todo en uno, integrada y certificada que proporciona no solo la radio inalámbrica, sino también un procesador integrado con interfaces para conectarse con varios periféricos. En este proyecto se utiliza para mandar los datos de la señal ECG al servidor por Wifi.
